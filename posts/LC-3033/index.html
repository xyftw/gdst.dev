🔗 🟢 3033. Modify the Matrix 1181

tags: Weekly Contest 384 陣列(Array) 矩陣(Matrix)

題意

給定一個大小為 m×nm \times nm×n 的整數矩陣 matrix\text{matrix}matrix，下標從 000 開始。新建一個大小同樣為 m×nm \times nm×n 的整數矩陣 answer\text{answer}answer，使 answer\text{answer}answer 與 matrix\text{matrix}matrix 相等，接著將其中每個值為 −1-1−1 的元素替換為所在 直行(column) 的 最大 元素。

返回矩陣 answer\text{answer}answer。

約束條件

  • m==matrix.lengthm == \text{matrix.length}m==matrix.length
  • n==matrix[i].lengthn == \text{matrix[i].length}n==matrix[i].length
  • 2≤m,n≤502 \leq m, n \leq 502≤m,n≤50
  • −1≤matrix[i][j]≤100-1 \leq \text{matrix[i][j]} \leq 100−1≤matrix[i][j]≤100
  • 每個 column 至少包含一個非負整數。

思路：維護每個 column 的最大值

由於需要將每個值為 −1-1−1 的元素替換為所在 column 的最大值，因此我們需要先使用一個大小為 nnn 的陣列 mxmxmx 來維護每個 column 的最大值。

接著重新遍歷整個矩陣，對於每個元素，如果它的值是 −1-1−1，就將其替換為該 column 的最大值；否則保留原值。

複雜度分析

  • 時間複雜度：O(m×n)O(m \times n)O(m×n)，需要遍歷整個矩陣兩次，一次是計算每個 column 的最大值，一次是生成新的矩陣 answer\text{answer}answer。
  • 空間複雜度：O(n)O(n)O(n)，用於存儲 mxmxmx 陣列。

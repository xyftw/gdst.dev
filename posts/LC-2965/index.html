🔗 🟢 2965. Find Missing and Repeated Values 1245

tags: Weekly Contest 376 計數(Counter) 雜湊表(Hash Table) 位運算(Bit Manipulation) 數學(Math)

題意

給定一個下標從 000 開始的二維整數矩陣 gridgridgrid，大小為 n×nn \times nn×n，其中的值皆在 [1,n2][1, n^2][1,n2] 範圍內。

gridgridgrid 中除了 aaa 出現 兩次，bbb 缺失 之外，每個整數都 恰好出現一次。以長度為 222 的列表形式返回重複的數字 aaa 和缺失的數字 bbb ，需按照 [a,b][a, b][a,b] 的順序返回。

思路

方法一：計數(Counter) / 雜湊表(Hash Table)

根據題意 grid[i][j]grid[i][j]grid[i][j] 的值域為 [1,n2][1, n^2][1,n2]，因此可以使用一個長度為 n2+1n^2+1n2+1 的陣列或雜湊表 cntcntcnt 來計算每個數字出現的次數，最後找出出現兩次的數字和沒有出現的數字即可。

複雜度分析

  • 時間複雜度 O(n2)\mathcal{O}(n^2)O(n2)
  • 空間複雜度 O(n2)\mathcal{O}(n^2)O(n2)

方法二：位運算(Bit Manipulation)

將 gridgridgrid 視為長度為 n2n^2n2 的一維陣列，若在 gridgridgrid 中額外添加 [1,n2][1, n^2][1,n2] 的所有數字，則在 [1,n2][1, n^2][1,n2] 中重複的數字 aaa 會出現三次、缺失的數字 bbb 會出現一次，其餘數字皆會出現兩次。

如此便可以將此問題轉換為 260. Single Number III 的問題，使用位運算來找出 aaa 和 bbb。過程可以參考該題的 解題紀錄 ，這裡不再贅述。

但需注意，在該題中返回的順序是任意的，而這題要求返回的順序是 [a,b][a, b][a,b]，因此在最後需要檢查 aaa 是否在 gridgridgrid 中，若不在則將 aaa 和 bbb 交換。

複雜度分析

  • 時間複雜度 O(n2)\mathcal{O}(n^2)O(n2)
  • 空間複雜度 O(1)\mathcal{O}(1)O(1)

方法三：數學(Math)

令 m=n2m = n^2m=n2 ，gridgridgrid 中的所有元素和為 s1s_1s1​ ，所有元素的平方和為 s2s_2s2​ 。若將 [1,m][1, m][1,m] 的所有數字相加，則總和為 ∑i=1n2i=m(m+1)2\sum_{i=1}^{n^2} i = \frac{m(m+1)}{2}∑i=1n2​i=2m(m+1)​；將 [1,m][1, m][1,m] 的所有數字的平方相加，則總和為 ∑i=1n2i2=m(m+1)(2m+1)6\sum_{i=1}^{n^2} i^2 = \frac{m(m+1)(2m+1)}{6}∑i=1n2​i2=6m(m+1)(2m+1)​。

由於在 gridgridgrid 中 aaa 出現兩次，bbb 缺失，因此 s1s1s1 和 s2s2s2 會比 m(m+1)2\frac{m(m+1)}{2}2m(m+1)​ 和 m(m+1)(2m+1)6\frac{m(m+1)(2m+1)}{6}6m(m+1)(2m+1)​ 多 a−ba - ba−b 和 a2−b2a^2 - b^2a2−b2 ，即 a−b=s1−m(m+1)2a - b = s_1 - \frac{m(m+1)}{2}a−b=s1​−2m(m+1)​ 和 a2−b2=s2−m(m+1)(2m+1)6a^2 - b^2 = s_2 - \frac{m(m+1)(2m+1)}{6}a2−b2=s2​−6m(m+1)(2m+1)​ ，分別令為 d1d1d1 和 d2d2d2。

聯立 d1=a−bd1 = a - bd1=a−b 和 d2=a2−b2=(a+b)(a−b)d2 = a^2 - b^2 = (a + b)(a - b)d2=a2−b2=(a+b)(a−b) ，即可解出 aaa 和 bbb。

  • 兩式相除得到 a+b=d2d1a + b = \frac{d2}{d1}a+b=d1d2​
  • d2d1+d1=2a⇒a=d2d1+d12\frac{d2}{d1} + d1 = 2a \Rightarrow a = \frac{\frac{d2}{d1} + d1}{2}d1d2​+d1=2a⇒a=2d1d2​+d1​
  • d2d1−d1=2b⇒b=d2d1−d12\frac{d2}{d1} - d1 = 2b \Rightarrow b = \frac{\frac{d2}{d1} - d1}{2}d1d2​−d1=2b⇒b=2d1d2​−d1​

複雜度分析

  • 時間複雜度 O(n2)\mathcal{O}(n^2)O(n2)，計算 s1s1s1 和 s2s2s2 需要遍歷整個 gridgridgrid
  • 空間複雜度 O(1)\mathcal{O}(1)O(1)

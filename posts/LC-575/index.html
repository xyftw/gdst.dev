🔗 🟢 575. Distribute Candies

tags: 貪心(Greedy) 集合(Set)

題意

給定一個長度為 nnn 的整數陣列 candyTypecandyTypecandyType，表示每個糖果的類型。Alice 想要在吃掉 n/2n / 2n/2 顆糖果的情況下，吃到最多不同種類的糖果。

返回 Alice 在只吃掉 n/2n / 2n/2 顆糖果的情況下，可以吃到糖的最多種類數，nnn 為偶數。

思路：貪心(Greedy) + 集合(Set)

若糖果種類數量為 kkk ，考慮以下兩種情況：

  1. k>n2k > \frac{n}{2}k>2n​，由於 Alice 最多只能吃 n/2n / 2n/2 種糖果，因此能吃到的最多種類數即為 n2\frac{n}{2}2n​。
  2. k≤n2k \leq \frac{n}{2}k≤2n​，則 Alice 最多能吃到 kkk 種糖果。

故答案為 min⁡(k,n2)\min(k, \frac{n}{2})min(k,2n​) ，其中 kkk 為糖果種類數量，可以使用 set 來計算。

複雜度分析

  • 時間複雜度：O(n)O(n)O(n)
  • 空間複雜度：O(n)O(n)O(n)

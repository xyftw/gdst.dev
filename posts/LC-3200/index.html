🔗 🟢 3200. Maximum Height of a Triangle 1451

tags: Weekly Contest 404 陣列(Array) 枚舉(Enumeration) 數學(Math)

題意

給定兩個整數 redredred 和 blueblueblue，分別表示紅色和藍色球的數量。你需要將這些球排列成一個三角形，使得第 111 行有 111 個球，第 222 行有 222 個球，第 333 行有 333 個球，以此類推。

同一行的所有球應該是 相同 的顏色，且相鄰行應該是 不同 的顏色。

返回所能達到的三角形的 最大高度。

約束條件

  • 1≤red,blue≤1001 \leq red, blue \leq 1001≤red,blue≤100

思路：枚舉高度 / 數學計算

方法一：枚舉高度

注意到紅色和藍色球數量最多只有 100100100 個，而三角形的最高高度滿足 1+2+3+...+h=h(h+1)2≤(red+blue)1 + 2 + 3 + ... + h = \frac{h(h + 1)}{2} \leq (red + blue)1+2+3+...+h=2h(h+1)​≤(red+blue)，因此三角形的高度最多為 200≈14\sqrt{200} \approx 14200​≈14 行。

因此我們可以枚舉三角形的高度，並檢查是否能夠構建出對應的三角形結構。並且由於三角形的高度是遞增的，因此我們可以從高度 111 開始枚舉，直到無法構建出對應的三角形結構為止。

具體來說，對於每一個高度 hhh，我們需要檢查是否能夠使用給定的紅色和藍色球來構建出對應的三角形結構。定義一個函數 check(x, y)，用來計算當第一行為紅色球，且紅色球數量為 xxx，藍色球數量為 yyy 時，能夠達到的最大高度。

  • 如果當前行數為奇數，則使用紅色球，並檢查紅色球是否足夠。如果足夠，則減去該行所需的紅色球數量；否則，停止檢查。
  • 如果當前行數為偶數，則使用藍色球，並檢查藍色球是否足夠。如果足夠，則減去該行所需的藍色球數量；否則，停止檢查。
  • 逐行增加高度，直到不能滿足條件為止。

最後，我們需要分別考慮兩種顏色起始的情況，取其中能夠構建的更高的三角形高度作為最終結果。

複雜度分析

  • 時間複雜度：O(h)=O(n+m)\mathcal{O}(h) = \mathcal{O}(\sqrt{n + m})O(h)=O(n+m​)，其中 hhh 是三角形的高度，nnn 和 mmm 分別是紅色和藍色球的數量。
  • 空間複雜度：O(1)\mathcal{O}(1)O(1)，只使用了常數額外空間。

方法二：數學計算

注意到在方法一中的複雜度計算中，我們使用了數學公式來計算三角形的高度。類似地，我們也可以使用數學公式來直接計算出能夠組成的最大高度，而不需要逐步嘗試每一層的可能性。

同樣地，我們定義一個函數 check(x, y)，用來計算當紅色球在奇數層，藍色球在偶數層時，且紅色球數量為 xxx，藍色球數量為 yyy 時，能夠達到的最大高度。並透過數學公式直接計算出能夠組成的最大高度，以下是詳細的計算步驟：

  1. 計算紅色球能支撐的最大奇數層數（t1t_1t1​）：
    • 每個奇數層需要的紅色球數量形成一個等差數列：1,3,5,...,(2×t1−1)1, 3, 5, ..., (2 \times t_1 - 1)1,3,5,...,(2×t1​−1)。這個數列的總和可以表示為：

    1+3+5+...+(2×t1−1)=t12≤x1 + 3 + 5 + ... + (2 \times t_1 - 1) = t_1^2 \leq x 1+3+5+...+(2×t1​−1)=t12​≤x

    • 因此，我們可以利用紅色球的總數量 xxx，通過取其平方根來得到最大可能的奇數層數：

    t1=⌊x⌋t_1 = \lfloor \sqrt{x} \rfloor t1​=⌊x​⌋

  2. 計算藍色球能支撐的最大偶數層數（t2t_2t2​）：
    • 每個偶數層需要的藍色球數量形成一個等差數列：2,4,6,...,(2×t2)2, 4, 6, ..., (2 \times t_2)2,4,6,...,(2×t2​)。這個數列的總和可以表示為：

    S=2+4+6+...+(2×t2)=t2×(t2+1)≤yS = 2 + 4 + 6 + ... + (2 \times t_2) = t_2 \times (t_2 + 1) \leq y S=2+4+6+...+(2×t2​)=t2​×(t2​+1)≤y

    • 因此，我們可以利用藍色球的總數量 yyy，通過解二次方程來得到最大可能的偶數層數：

    t22+t2≤yt_2^2 + t_2 \leq y t22​+t2​≤y

    • 解這個二次方程，我們可以得到：

    t2=⌊−1+1+4×y2⌋t_2 = \lfloor \frac{-1 + \sqrt{1 + 4 \times y}}{2} \rfloor t2​=⌊2−1+1+4×y​​⌋

    • 如果 t1>t2t_1 > t_2t1​>t2​，那麼可以構成 t2+1t_2 + 1t2​+1 行紅色球和 t2t_2t2​ 行藍色球，因此最大高度為 2×t2+12 \times t_2 + 12×t2​+1。
    • 否則，可以構成 t1t_1t1​ 行紅色球和 t1t_1t1​ 行藍色球，因此最大高度為 2×t12 \times t_12×t1​。

同樣地，我們需要分別考慮兩種顏色起始的情況，取其中能夠構建的更高的三角形高度作為最終結果。

複雜度分析

  • 時間複雜度：O(1)\mathcal{O}(1)O(1)，只進行了常數次數的計算。
    • sqrt 函數的時間複雜度可以視為 O(1)\mathcal{O}(1)O(1)。
  • 空間複雜度：O(1)\mathcal{O}(1)O(1)，只使用了常數額外空間。

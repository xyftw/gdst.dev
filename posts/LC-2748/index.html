LeetCode 🟢 2748. Number of Beautiful Pairs

🔗 🟢 2748. Number of Beautiful Pairs 1301

tags: Weekly Contest 351 數學(Math) 最大公因數(GCD) 暴力法(Brute Force) 計數(Counting) 雜湊表(Hash Table)

題意

給定一個下標從 000 開始的整數陣列 numsnumsnums。 若存在一對索引 (i,j)(i, j)(i,j)，滿足 0≤i<j<nums.length0 \leq i < j < \text{nums.length}0≤i<j<nums.length，且 nums[i]\text{nums}[i]nums[i] 的第一位數字與 nums[j]\text{nums}[j]nums[j] 的最後一位數字互質的話，就稱這對索引為 美麗對(beautiful pairs) 。

返回 numsnumsnums 中美麗對的總數。

對於兩個整數 xxx 和 yyy ，若沒有大於 111 的整數能同時整除它們，則稱它們為互質。換句話說，若 xxx 和 yyy 的最大公因數為 111，則它們互質。

約束條件

  • 2≤n=nums.length≤1002 \leq n = \text{nums.length} \leq 1002≤n=nums.length≤100
  • 1≤nums[i]≤99991 \leq \text{nums}[i] \leq 99991≤nums[i]≤9999
  • nums[i]mod  10≠0\text{nums}[i] \mod 10 \neq 0nums[i]mod10=0

思路

gcd 函數是計算兩個整數的最大公因數，可以使用遞迴或迭代的方式來實現，這裡使用遞迴的方式，定義 gcd 函數如下：

  • 若 b=0b = 0b=0，則 gcd⁡(a,b)=a\gcd(a, b) = agcd(a,b)=a，
  • 否則 gcd⁡(a,b)=gcd⁡(b,amod  b)\gcd(a, b) = \gcd(b, a \mod b)gcd(a,b)=gcd(b,amodb)。

由於需要計算一個數字的第一個數字和最後一個數字，可以使用除法和取餘數的方式來實現：

  • 取出一個數字 xxx 的第一個數字，可以將 xxx 除以 101010 直到 xxx 小於 101010 為止，最後的 xxx 即為第一個數字。
  • 取出一個數字 xxx 的最後一個數字，可以將 x mod 10x \bmod 10xmod10 即可。

方法一：暴力法(Brute Force)

由於 n≤100n \leq 100n≤100，因此可以使用暴力法來解決這個問題。使用兩個迴圈來列舉所有可能的索引組合 (i,j)(i, j)(i,j)，然後檢查是否這對索引構成一個美麗對。具體來說，我們對 nums[i]\text{nums}[i]nums[i] 的第一位數字 yyy 和 nums[j]\text{nums}[j]nums[j] 的最後一位數字 xxx 做最大公因數的檢查，若等於 111 則代表兩個數字互質，算作一個美麗對，將答案 ansansans 加 1。

複雜度分析

  • 時間複雜度：O(n2log⁡M)\mathcal{O}(n^2 \log M)O(n2logM) ，其中 nnn 是陣列 numsnumsnums 的長度， MMM 是陣列 numsnumsnums 中的最大值。
    • 遍歷所有可能的 (i,j)(i, j)(i,j) 對需要 O(n2)\mathcal{O}(n^2)O(n2) 的時間。
    • 計算 nums[i]\text{nums}[i]nums[i] 的第一位數字 yyy 需要 O(log⁡M)\mathcal{O}(\log M)O(logM) 的時間。
    • 由於 x,y<10x, y < 10x,y<10 ，因此計算最大公因數的時間可以視為常數時間。
  • 空間複雜度：O(1)\mathcal{O}(1)O(1) 。

方法二：計數(Counting)

從方法一中可以發現，對於 nums[j]\text{nums}[j]nums[j] 的最後一位數字 xxx ，只需要考慮前面有多少數字的第一個數字 yyy 與 xxx 互質即可。

因此可以使用一個長度為 101010 的計數陣列 cntcntcnt 來記錄每個數字的出現次數。對於每一個數字 xxx，我們找出它的最後一位數字 x%10x \% 10x%10，然後和 111 到 999 之間的數字進行最大公因數檢查，若為 111 則意味兩數互質，可以構成美麗對，將 cntcntcnt 中對應的值加到答案 ansansans 上。

最後，我們更新 cntcntcnt 陣列，將 xxx 的第一個數字的計數增加 1。

複雜度分析

  • 時間複雜度：O(n)\mathcal{O}(n)O(n) ，其中 nnn 是陣列 numsnumsnums 的長度。
  • 空間複雜度：O(10)=O(1)\mathcal{O}(10) = \mathcal{O}(1)O(10)=O(1) 。

🔗 UVA-11934 Magic Formula

tags: 暴力法(Brute Force)

範例程式碼已於 UVA、瘋狂程設(CPE) 上皆測試通過。

題意

題目給出一個二次函數 f(x)=ax2+bx+cf(x) = ax^2 + bx + cf(x)=ax2+bx+c，以及一個除數 ddd 和一個上限 LLL。要求計算從 f(0)f(0)f(0) 到 f(L)f(L)f(L) 中，有多少個函數值能被 ddd 整除。

約束條件

  • −1000≤a,b,c≤1000-1000 \leq a, b, c \leq 1000−1000≤a,b,c≤1000
  • 1<d<10000001 < d < 10000001<d<1000000
  • 0≤L<10000 \leq L < 10000≤L<1000

思路：暴力法(Brute Force)

枚舉 [0,L][0, L][0,L] 中的每個數 xxx，計算 f(x)f(x)f(x) 的值，並檢查是否能被 ddd 整除。若能被整除，則將答案 ansansans 加一。

複雜度分析

  • 時間複雜度：O(L)\mathcal{O}(L)O(L)
  • 空間複雜度：O(1)\mathcal{O}(1)O(1)

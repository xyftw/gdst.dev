🔗 🟡 1653. Minimum Deletions to Make String Balanced 1793

tags: Biweekly Contest 39 堆疊(Stack) 字串(String) 動態規劃(DP)

題意

給定一個只包含字元 'a' 和 'b' 的字串 sss。

若 不存在 任何下標對 (i,j)(i, j)(i,j) 满足 i<ji < ji<j 且 s[i]=′b′s[i] = 'b's[i]=′b′ 且 s[j]=′a′s[j] = 'a's[j]=′a′ ，則稱 sss 是 平衡 的。

為使 sss 平衡，你可以刪除 sss 中任意數量的字元，返回使 sss 平衡的最小刪除次數。

思路：動態規劃(DP)

根據題意，平衡子序列的定義即為所有 aaa 皆在 bbb 之前。

我們可以將問題轉換成求刪除某些字元後，最大的平衡子序列長度，如此便可以使用 動態規劃(DP) 來解決這個問題。

令 dp[i][0/1]dp[i][0/1]dp[i][0/1] 表示從 s[0:i]s[0:i]s[0:i] 刪除某些元素後，使得所有 aaa 皆在 bbb 之前的最長子序列長度，且最後一個字元是 aaa 或 bbb。則有以下轉移方程：

  • 如果 s[i−1]=as[i-1] = as[i−1]=a，則 dp[i][0]=dp[i−1][0]+1dp[i][0] = dp[i-1][0] + 1dp[i][0]=dp[i−1][0]+1。
    • 由於 aaa 必須在 bbb 之前，只能從最後一個字元是 aaa 的情況轉移。
  • 如果 s[i−1]=bs[i-1] = bs[i−1]=b，則 dp[i][1]=max⁡(dp[i−1][0],dp[i−1][1])+1dp[i][1] = \max(dp[i-1][0], dp[i-1][1]) + 1dp[i][1]=max(dp[i−1][0],dp[i−1][1])+1。
    • 可以從最後一個字元是 aaa 或 bbb 的情況轉移。

注意到轉移時只與前一個狀態有關，因此可以進一步優化空間複雜度為 O(1)O(1)O(1) ，分別用 aaa 和 bbb 來表示最後一個字元是 aaa 或 bbb 的情況。

最終對兩者取 max⁡\maxmax 即可得到最大平衡子序列長度，故所求的刪除次數為 n−max(a,b)n - max(a, b)n−max(a,b) 。

複雜度分析

  • 時間複雜度：O(n)\mathcal{O}(n)O(n)，其中 nnn 是字串 sss 的長度。
  • 空間複雜度：O(1)\mathcal{O}(1)O(1)。

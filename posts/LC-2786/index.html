🔗 🟡 2786. Visit Array Positions to Maximize Score 1733

tags: Biweekly Contest 109 動態規劃(DP)

題意

給定一個下標從 000 開始的整數陣列 numsnumsnums 和一個正整數 xxx。

你一開始在陣列的 000 號位置，可以根據以下規則到其他位置：

  • 如果你現在在 iii 號位置，可以移動到任何滿足 i<ji < ji<j 的位置 jjj。
  • 對於你到達的每個位置 iii，你可以獲得分數 nums[i]nums[i]nums[i]。
  • 如果你從位置 iii 移動到位置 jjj ，但 nums[i]nums[i]nums[i] 和 nums[j]nums[j]nums[j] 的奇偶性不同，那麼你將失去分數 xxx。

你可以移動到任何 jjj 號位置,只要 i<ji < ji<j。

返回能夠獲得的 最大 總分。

請注意，一開始你有 nums[0]nums[0]nums[0] 分。

約束條件

  • 2≤nums.length≤1052 \leq nums.length \leq 10^52≤nums.length≤105
  • 1≤nums[i],x≤1061 \leq nums[i], x \leq 10^61≤nums[i],x≤106

思路：動態規劃(DP)

寫法一

令 dp[i][p]dp[i][p]dp[i][p] 表示考慮到第 iii 個位置，最後一個位置的奇偶性為 ppp (0 表示偶數，1 表示奇數) 時，能獲得的最大分數。

當從位置 i−1i-1i−1 到達位置 iii 時，我們有兩個選擇：

  1. 如果 nums[i−1]nums[i-1]nums[i−1] 和 nums[i]nums[i]nums[i] 的奇偶性相同 (ppp 相同)，由於 nums[i]nums[i]nums[i] 為正整數，選擇走到位置 iii 會使分數增加，因此 dp[i][p]dp[i][p]dp[i][p] 可以從 dp[i−1][p]+nums[i]dp[i-1][p] + nums[i]dp[i−1][p]+nums[i] 獲得。
  2. 如果奇偶性不同，則 dp[i][p]dp[i][p]dp[i][p] 可以從 dp[i−1][1−p]+nums[i]−xdp[i-1][1-p] + nums[i] - xdp[i−1][1−p]+nums[i]−x 獲得，其中 1−p1-p1−p 表示與 ppp 相反的奇偶性。

故可以得到狀態轉移方程：

  • dp[i][p]=max⁡(dp[i−1][p]+nums[i],dp[i−1][1−p]+nums[i]−x)dp[i][p] = \max(dp[i-1][p] + nums[i], dp[i-1][1-p] + nums[i] - x)dp[i][p]=max(dp[i−1][p]+nums[i],dp[i−1][1−p]+nums[i]−x)
  • dp[i][1−p]=dp[i−1][1−p]dp[i][1-p] = dp[i-1][1-p]dp[i][1−p]=dp[i−1][1−p] ，表示當前位置的另一種奇偶性的最大得分不變。

具體做法如下：

  1. 初始化 dpdpdp 陣列，dp[i][p]dp[i][p]dp[i][p] 表示考慮到第 iii 個位置，最後一個位置的奇偶性為 ppp 時，能獲得的最大分數。
    • 由於從位置 000 開始，所以 dp[0][nums[0] mod 2]=nums[0]dp[0][nums[0] \bmod 2] = nums[0]dp[0][nums[0]mod2]=nums[0] 。
    • dpdpdp 陣列的其他值初始化為負無窮，表示未達到該位置。
    • 初始化 ans=nums[0]\text{ans} = nums[0]ans=nums[0]。
  2. 遍歷從 111 到 n−1n-1n−1 的位置 iii。對於每個 iii，根據 nums[i]nums[i]nums[i] 的奇偶性 p=nums[i] mod 2p = nums[i] \bmod 2p=nums[i]mod2 更新 dp[i][p]dp[i][p]dp[i][p] 和 dp[i][1−p]dp[i][1-p]dp[i][1−p]。
  3. 每一步都更新最大分數 ansansans 為 max⁡(ans,dp[i][p])\max(\text{ans}, dp[i][p])max(ans,dp[i][p])。
  4. 最後返回 ansansans 即可。

複雜度分析

  • 時間複雜度：O(n)\mathcal{O}(n)O(n)，其中 nnn 是陣列 numsnumsnums 的長度，只需要遍歷一次陣列。
  • 空間複雜度：O(n)\mathcal{O}(n)O(n)，使用了長度為 n×2n \times 2n×2 的二維陣列 dpdpdp。

寫法二：空間優化

注意到在寫法一中，dp[i][p]dp[i][p]dp[i][p] 的值只和前一個狀態的 dp[i−1][p]dp[i-1][p]dp[i−1][p] 和 dp[i−1][1−p]dp[i-1][1-p]dp[i−1][1−p] 有關，且 dp[i][1−p]dp[i][1-p]dp[i][1−p] 的值在每一步都不變，因此可以對空間進行優化，只使用一個長度為 222 的陣列 fff 來存儲 dp[i][p]dp[i][p]dp[i][p] 和 dp[i][1−p]dp[i][1-p]dp[i][1−p] 的值。

  • 時間複雜度：O(n)\mathcal{O}(n)O(n)，其中 nnn 是陣列 numsnumsnums 的長度，只需要遍歷一次陣列。
  • 空間複雜度：O(1)\mathcal{O}(1)O(1)，只使用了長度為 222 的陣列 fff。
